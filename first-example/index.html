Selecting by ID

The getElementById method of the document object is the most efficient way to select an element if it has an id assigned to it.

Selecting by name

You can also use getElementsByName for nodes with a name attribute, but since there can be multiple elements with the same name, getElementsByName will return a NodeList object that acts as an array of elements in the case of multiple matches.



Selecting by CSS Selector

The querySelector method allows us a way to use CSS selectors to select DOM elements. You can pass any valid CSS selector into this method, which selects the first matching element.
